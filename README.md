🌾 AgroLanka

AgroLanka is a full-stack web application developed to modernize agricultural management by connecting farmers, suppliers, and customers through a single digital platform. The system streamlines farm operations, product management, and user interactions with an intuitive and responsive interface.

🚀 Features
👤 User authentication and authorization
🌱 Farm and crop management
🛒 Product listing and management
📦 Order management
💳 Secure payment integration
📊 Dashboard for managing activities
📱 Responsive user interface

⚙️ Installation
Clone the repository
git clone https://github.com/IT24102102/ITP_11_AgroLanka.git
Install dependencies

Frontend

cd client
npm install

Backend

cd server
npm install
Configure environment variables

Create a .env file in the server directory and add the required environment variables.

Start the application

Backend

npm start

Frontend

npm start
